# JavaScript Style Guide

## File Naming Convention

All files must use **kebab-case**.

### Rules

- Use lowercase letters only
- Separate words with hyphens (`-`)
- No underscores, spaces, or camelCase

### Examples

| Correct              | Incorrect           |
| -------------------- | ------------------- |
| `button.tsx`         | `Button.tsx`        |
| `date-picker.tsx`    | `DatePicker.tsx`    |
| `use-toggle.ts`      | `useToggle.ts`      |
| `form-validation.ts` | `formValidation.ts` |

### Directory Names

Directories also follow kebab-case:

| Correct        | Incorrect     |
| -------------- | ------------- |
| `components/`  | `Components/` |
| `date-picker/` | `DatePicker/` |
| `use-hooks/`   | `useHooks/`   |
